# API 变更日志

## [1.0] - 2026-06-23

### 新增
- 创建全部API接口定义，共22个接口
- 模块分布：
  - 用户模块：3个（注册、登录、退出）
  - 图书模块：2个（列表含搜索分类、详情）
  - 购物车模块：5个（查看、加入、修改数量、删除商品、清空）
  - 订单模块：5个（提交订单、我的订单、模拟支付、取消订单、确认收货）
  - 后台管理模块：7个（图书列表、添加、编辑、删除、上下架、订单列表、发货）

### 技术决策
- 认证方式：Session（Cookie），不使用JWT
- 响应格式统一：`{ "code": 200, "message": "success", "data": {} }`
- 分页参数统一：`page`（从1开始）、`page_size`（默认12）
- 图片上传：`multipart/form-data`，存储在本地`/static/uploads/`
